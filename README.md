## Pythonと外部ライブラリ
ソースコードを実行するには、下記のソフトウェアが必要です。

* Python 3.x
* NumPy
* Matplotlib

※Pythonのバージョンは、3系を利用します。

## 実行方法

任意のJupyter Notebook環境で各章のNotebookを開き、Pythonコマンドを実行します。
